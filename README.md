# Portal de Aprovações de Solicitações de Compra

Um portal web moderno para gerenciar e aprovar solicitações de compra de forma eficiente e centralizada.

## 🎯 Sobre o Projeto

O **Portal de Aprovações** é uma aplicação que permite organizações gerenciarem o fluxo de aprovações de solicitações de compra. A plataforma oferece uma interface intuitiva para submeter, rastrear e aprovar solicitações com base em diferentes níveis de autorização.

## 🚀 Tecnologias Utilizadas

- **React 18** - Biblioteca JavaScript para construção de interfaces
- - **TypeScript** - Linguagem de programação com tipagem estática
  - - **Vite** - Build tool rápido para desenvolvimento moderno
    - - **ESLint** - Ferramenta para análise de código
     
      - ## 📋 Funcionalidades
     
      - - Submissão de solicitações de compra
        - - Sistema de aprovação em múltiplos níveis
          - - Rastreamento de status das solicitações
            - - Interface responsiva e intuitiva
              - - Validação de dados em tempo real
               
                - ## 🛠️ Como Começar
               
                - ### Pré-requisitos
               
                - - Node.js 16+
                  - - npm ou yarn
                   
                    - ### Instalação
                   
                    - ```bash
                      # Clonar o repositório
                      git clone https://github.com/guialvesmartins/portal-aprovacao.git

                      # Entrar no diretório
                      cd portal-aprovacao

                      # Instalar dependências
                      npm install
                      ```

                      ### Desenvolvimento

                      ```bash
                      # Iniciar o servidor de desenvolvimento
                      npm run dev
                      ```

                      ### Build para Produção

                      ```bash
                      # Compilar para produção
                      npm run build
                      ```
